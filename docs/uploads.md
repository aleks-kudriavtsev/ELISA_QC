# Загрузка файлов

## CSV

Поддерживаются CSV-файлы с обязательными колонками:

- `Well`
- `OD`
- `Wavelength`
- `SampleID`

Файлы следует хранить в каталоге `csv/`, при наличии разделяйте сырые и обработанные данные (например, `csv/raw/` и `csv/processed/`).

## Изображения

Допускаются изображения в форматах PNG и JPEG. Для каждого изображения требуется описание с указанием типа прибора. Изображения следует хранить в каталоге `images/` (при необходимости разделяя сырые и обработанные данные).

## Привязка к ExperimentRun

Каждый загружаемый файл должен быть привязан к конкретному `ExperimentRun` и включать ссылку на идентификатор запуска эксперимента.

## Ошибки и валидация

При обработке загрузок необходимо выполнять проверку:

- битые или неполные файлы отклоняются;
- несовпадение формата (например, не-CSV для табличных данных) приводит к ошибке;
- отсутствие обязательных колонок в CSV вызывает ошибку валидации.
