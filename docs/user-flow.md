# Пользовательский сценарий: Telegram mini-app для ИФА

## Последовательность работы
1. Вход в Telegram mini-app.
2. Выбор шаблона ИФА или создание плана.
3. Ввод параметров (концентрации, объемы, контролли).
4. Прохождение чек-листов по шагам.
5. Ввод фактических параметров.
6. Загрузка CSV/изображений.
7. Просмотр итоговой сводки и подтверждение завершения.

## Ключевые боли и риски
- Ошибки дозировок (неверные концентрации/объемы, перепутанные контролли).
- Пропуск шагов чек-листа или подтверждений, приводящий к неполному протоколу.
